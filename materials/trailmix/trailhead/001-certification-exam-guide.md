# Certification Exam Guide — Platform Integration Architect

**Source:** https://trailhead.salesforce.com/help?article=Salesforce-Certified-Platform-Integration-Architect-Exam-Guide  
**Saved:** browser extraction (logged-in session)

---

## About the Exam

- **Content:** 60 multiple-choice + up to 5 unscored questions
- **Time:** 105 minutes
- **Passing score:** 67%
- **Fee:** US$400 (retake US$200)
- **Delivery:** testing center or online proctored

## Exam Outline (weights)

| Section | Weight |
|---------|--------|
| Evaluate the Current System Landscape | 8% |
| Evaluate Business Needs | 11% |
| Translate Needs to Integration Requirements | 22% |
| Design Integration Solutions | 28% |
| Build Solution | 23% |
| Maintain Integration | 8% |

### Evaluate the Current System Landscape (8%)
- Identify current system landscape, standards, limitations, boundaries, protocols
- Analyze constraints/pain-points for business requirements
- Evaluate authentication and authorization needs

### Evaluate Business Needs (11%)
- Functional and non-functional integration requirements
- Classify data: Confidential / Secure / Public
- CRM success factors as integration requirements
- Business growth and regulatory factors

### Translate Needs to Integration Requirements (22%)
- Create inventory of systems and integration patterns from landscape diagram
- Evaluate system and process constraints
- Integration security / auth / authorization requirements
- Performance needs (volumes, response times, latency)

### Design Integration Solutions (28%)
- Integration pattern for business requirements
- Components for solution
- Trade-offs, limitations, constraints
- Appropriate Salesforce API(s)
- Standards, components, techniques, security mechanisms

### Build Solution (23%)
- API design (Salesforce as provider and consumer)
- Outbound call considerations to external systems
- Scalable solution design
- Error handling for integration options
- Security for inbound/outbound integrations
- Resilience for system updates

### Maintain Integration (8%)
- Performance monitoring for integrations
- Error handling, escalation, recovery procedures
- Reporting for integration monitoring

## Candidate background (typical)

- 1–2 years Salesforce Platform Integration Architecture
- 2–3 years Salesforce admin/developer
- 1+ year enterprise integration solutions

## Candidate should know

- High-performing, scalable, secure integrations
- Integration architecture blueprint
- Integration patterns and appropriate APIs
- Platform limits, security, error handling

## Candidate NOT expected to know

- Non-Salesforce DB concepts
- MDM tools configuration
- Lightning dev expertise
- Programming languages

## Maintenance

Annual maintenance badge on Trailhead required.
